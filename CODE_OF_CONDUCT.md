# Code of Conduct

## Our Pledge

We want `RLPlasticity` to be a welcoming, respectful, and technically serious open source project.

Contributors, maintainers, and community members are expected to act in ways that make collaboration safe, constructive, and sustainable.

## Expected Behavior

Examples of positive behavior include:

- showing empathy and patience
- giving clear, actionable technical feedback
- assuming good intent while being honest about problems
- respecting different backgrounds and experience levels
- focusing criticism on ideas, code, and evidence

## Unacceptable Behavior

Examples of unacceptable behavior include:

- harassment, insults, or personal attacks
- discriminatory language or conduct
- sustained hostility or intimidation
- publishing private information without permission
- bad-faith disruption of discussion or review

## Scope

This Code of Conduct applies in:

- GitHub issues
- pull requests
- discussions
- project communication spaces tied to this repository

## Reporting

If you experience or witness unacceptable behavior, please contact the maintainers privately.

Until a dedicated project contact is added, use the repository owner contact shown on GitHub and include:

- what happened
- where it happened
- links or screenshots if relevant

## Enforcement

Maintainers may remove comments, close discussions, or restrict participation when necessary to protect the project and its community.

## Attribution

This document is informed by common open source community standards, including the Contributor Covenant approach, but adapted to fit this repository's current size and governance.
